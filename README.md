# farmy

## Rodando a aplicação

### Servidor do jogo

```sh
$ cd server
$ npm install
$ npm start
```

### Cliente

```sh
$ cd client
$ npm install
$ npm start
```

Depois, em outro terminal:

```sh
$ cd client
$ npm run serve
```

O script `serve` subirá um servidor HTTP na porta 5000, servindo o diretório dist.
